# 编码系统

一个用于教育文本分析的编码系统，支持校准和编码两种模式。

## 功能特点

- 支持Excel文件的读写
- 提供直观的图形用户界面
- 支持校准和编码两种模式
- 可自定义API设置
- 实时显示处理进度
- 提供准确率统计

## 安装说明

1. 确保已安装Python 3.7或更高版本
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
