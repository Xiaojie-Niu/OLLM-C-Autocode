# 编码系统

一个用于教育文本分析的编码系统，支持校准和编码两种模式。

## 功能特点

- 支持Excel文件的读写
- 提供直观的图形用户界面
- 支持校准和编码两种模式
- 可自定义API设置
- 实时显示处理进度
- 提供准确率统计
